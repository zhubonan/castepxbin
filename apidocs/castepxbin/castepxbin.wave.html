castepxbin.wave#

Functions for handling wave function related data

Module Contents#

Classes#

WaveFunction

Class for handling the wave function

Functions#

coords_to_indices

Convert reciprocal lattice indexing to a grid based indices. This is allows reconstruct the wave function on to a reciprocal space grid.

coeff_to_recip

Convert compact coefficient representation to a full reciprocal grid representation for the wave function. The output data is on a 3D grid that is ready to be inverse FFTed.

API#

Convert reciprocal lattice indexing to a grid based indices. This is allows reconstruct the wave function on to a reciprocal space grid.

castepxbin.wave.coeff_to_recip(coeffs: numpy.ndarray, nwaves_at_kp: numpy.ndarray, grid_coords: numpy.ndarray, ngx: int, ngy: int, ngz: int)#

Convert compact coefficient representation to a full reciprocal grid representation for the wave function. The output data is on a 3D grid that is ready to be inverse FFTed.

Parameters:

coeffs – Plane wave coefficients, in the shape of (npw, nspinor, nb, nk, nspins)

Returns:

The plane wave coefficients on the grid, in the shape of (ngx, ngy, ngz, nspinor, nband, nk, nspins)
